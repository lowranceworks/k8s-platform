# MetalLB

Network load-balancer for bare-metal Kubernetes clusters. Provides LoadBalancer-type Service
support using BGP or Layer 2 protocols.

- **Upstream chart:** [metallb/metallb](https://artifacthub.io/packages/helm/metallb/metallb)
- **Documentation:** <https://metallb.io/>
- **GitHub:** <https://github.com/metallb/metallb>
