# External Secrets Operator

Syncs secrets from external providers (GCP Secret Manager, AWS Secrets Manager, Vault, etc.)
into Kubernetes Secrets. Keeps sensitive data out of Git.

- **Upstream chart:** [external-secrets/external-secrets](https://artifacthub.io/packages/helm/external-secrets-operator/external-secrets)
- **Documentation:** <https://external-secrets.io/latest/>
- **GitHub:** <https://github.com/external-secrets/external-secrets>
