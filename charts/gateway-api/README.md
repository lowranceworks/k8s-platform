# Gateway API

Role-oriented Kubernetes API for service networking. Provides expressive, extensible routing
for HTTP, TCP, and gRPC traffic as a successor to Ingress.

- **Upstream chart:** [kubernetes-sigs/gateway-api](https://artifacthub.io/packages/helm/gateway-api/gateway-api)
- **Documentation:** <https://gateway-api.sigs.k8s.io/>
- **GitHub:** <https://github.com/kubernetes-sigs/gateway-api>
