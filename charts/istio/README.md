# Istio

Service mesh for Kubernetes. Provides traffic management, security (mTLS), and observability
across microservices. This chart installs both the `base` CRDs and `istiod` control plane.

- **Upstream charts:** [istio/base](https://artifacthub.io/packages/helm/istio-official/base), [istio/istiod](https://artifacthub.io/packages/helm/istio-official/istiod)
- **Documentation:** <https://istio.io/latest/docs/>
- **GitHub:** <https://github.com/istio/istio>
